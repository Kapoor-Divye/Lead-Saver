# Lead-Saver
A simple yet powerful Chrome Extension that lets you save potential customer leads (like social media profiles or important links) with just one click.  Perfect for marketers, freelancers, and developers who want a quick way to store and manage leads while browsing.

📌 Description

Lead Saver helps you capture and store useful links instantly—without breaking your workflow.

Whether you're scouting clients on LinkedIn, Twitter, or any platform, this extension allows you to:

Save the current tab as a lead
Manually add links
View and manage saved leads anytime

All data is stored locally in your browser for fast and easy access.

🚀 Features
📥 Save current tab with one click
✍️ Manually input custom links
💾 Persistent storage using localStorage
🗑️ Delete all saved leads
📋 Clean and minimal UI
